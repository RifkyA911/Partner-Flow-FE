---

## English

A referral partner platform for Aice in the Tangerang Selatan region, built with Next.js for speed and SEO.

### Features
- Referral system with unique codes
- Partner dashboard
- Commission tracking
- Partner data management

### Getting Started
```
npm install
npm run dev
```

Open http://localhost:3000.

### Tech Stack
- Framework: Next.js
- Language: TypeScript
- Styling: Tailwind CSS
